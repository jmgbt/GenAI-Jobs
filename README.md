# GenAI Jobs

GenAI-Jobs

GenAI-Jobs is a personal project designed to structure and partially automate the job search process, from job offer collection to the generation of tailored CVs and cover letters in PDF format using generative AI.

The project deliberately avoids full automation of applications and instead focuses on building a controlled, transparent, and extensible pipeline, where automation supports human decision-making rather than replacing it.

- Centralize job offers from multiple sources
- Reduce the time spent on repetitive tasks during job search
- Generate customized CVs and cover letters based on a single source of truth
- Maintain full human control over whether and how applications are sent
- Build a robust, testable, and modular pipeline

ğŸ§  Why this project?

Modern job searching is caught between two extremes:
- mass application tools that sacrifice quality,
- and fully manual processes that are time-consuming and inefficient.

GenAI-Jobs explores a middle ground.

Instead of automatically sending applications at scale, the project automates what is repetitive and mechanical (offer collection, document adaptation, formatting), while keeping intentional human validation at the core of the process.

The project is both:
- a personal productivity tool,
- and an experimentation ground for applying generative AI in a realistic, constrained workflow.

ğŸ§  Core Design Principles

- Single source of truth: the CV content lives in cv_default.md
- No sensitive data committed: all secrets via environment variables
- Explicit control: no automatic submission of applications
- Incremental automation: each step is testable in isolation
- Readable outputs: clean, professional PDFs suitable for real applications

ğŸ¤– CV & Cover Letter Generation
CV
- Generated from cv_default.md
- Adapted to each job offer using AI
- Exported as a single-page, styled PDF
- Formatting handled programmatically (no LaTeX dependency)

Cover Letter
- Fully generated by AI (no static template)
- Uses both the job offer and the adapted CV as context
- Dedicated PDF layout (header, recipient block, body)
